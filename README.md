# Code that classifies the various usages of the Chinese complement ‘起来’ into five labels, makes predictions, and visualizes them using t-SNE.

## Labels
1: "Directional"
2: "Resultative"
3: "Completive"
4: "Inchoative"
5: "Discourse"

## Install requirements
pip install requirements.txt

## Analysis with ‘起来’
python main.py
